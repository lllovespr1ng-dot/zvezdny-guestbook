# Zvezdny Digital Guestbook

Статический прототип цифрового гостевого справочника для отеля "Звездный".

## Как открыть

Откройте `index.html` в браузере. При первом открытии гость вводит номер комнаты, он сохраняется в браузере и подставляется в заявки.

Для теста конкретного номера также можно добавить параметр:

```text
index.html?room=512
```

## Что уже есть

- мобильная главная страница;
- общий QR-сценарий с вводом номера комнаты;
- быстрые действия: Wi-Fi, проживание, завтрак, SPA;
- RU / EN переключатель;
- публичные контакты отеля;
- реальные данные по Wi-Fi, заезду/выезду, правилам проживания, уборке, парковке, ресторану, room service и SPA;
- заявки на услуги через модальное окно;
- форма обратной связи;
- отдельный макет QR-карточки: `qr-card.html`.

## QR-код

QR-код должен быть один общий для всех номеров и вести на опубликованный `index.html`.

Если guestbook уже опубликован, откройте:

```text
qr-card.html?url=https://your-public-url/index.html
```

Карточка покажет QR-код для указанного URL. После сканирования гость вводит номер комнаты самостоятельно.

## Что осталось подключить

- Telegram-чат, Google Forms или backend endpoint для отправки заявок;
- режим работы и условия доступа для пляжа, бассейнов и фитнес-клуба;
- финальный QR-код на опубликованный URL;
- согласованные финальные тексты отеля.
