# Цифровой гид отеля «Звездный»

Статический прототип цифрового гостевого справочника для отеля «Звездный».

## Как открыть

Откройте `index.html` в браузере. При первом открытии гость вводит номер комнаты, он сохраняется в браузере и подставляется в заявки.

Для теста конкретного номера также можно добавить параметр:

```text
index.html?room=512
```

## Что уже есть

- мобильная главная страница;
- общий QR-сценарий с вводом номера комнаты;
- быстрые действия: Wi-Fi, проживание, завтрак, СПА;
- RU / EN переключатель;
- публичные контакты отеля;
- реальные данные по Wi-Fi, заезду/выезду, правилам проживания, уборке, парковке, ресторану, обслуживанию номеров и СПА;
- заявки на услуги через всплывающее окно;
- форма обратной связи;
- отдельный макет QR-карточки: `qr-card.html`.
- печатный A6-макет для номера: `print-card.html`.
- статичный QR для опубликованной версии: `assets/qr-guestbook.svg`.

## QR-код

QR-код должен быть один общий для всех номеров и вести на опубликованную страницу `index.html`.

Если цифровой гид уже опубликован, откройте:

```text
qr-card.html?url=https://your-public-url/index.html
```

Карточка покажет QR-код для указанной ссылки. После сканирования гость вводит номер комнаты самостоятельно.

Для печати тестовой карточки откройте `print-card.html` и сохраните через браузер в PDF или распечатайте как A6.

## Что осталось подключить

- Telegram-чат, Google Forms или серверный обработчик для отправки заявок;
- режим работы и условия доступа для пляжа, бассейнов и фитнес-клуба;
- финальный QR-код на опубликованную ссылку;
- согласованные финальные тексты отеля.
